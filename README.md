# BHubCase
Esse projeto tem por objetivo criar uma API simplificada de gerenciamento e armazenamento de regras de negócio para um sistema de pagamentos. 

## Tecnologias utilizadas:
- Flask
- Python3
- VSCode
- Postman (testes)
